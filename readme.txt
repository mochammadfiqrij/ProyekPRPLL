Alur Program:

Keterangan:
Program ini merupakan program dari pengelolaan transaksi penjualan dari DebagsPro Shop.
Alur program ini akan menjelaskan bagaimana program ini dapat dijalankan, yang mana program ini menggunakan spesifikasi sebagai berikut:
1. Menggunakan bahasa pemrograman java dengan tools NetBeans IDE 8.0.2
2. Menggunakan DBMS MySQL menggunakan XAMPP.

Tata Cara membuka program:
1. Import terlebih dahulu database debagsku.sql ke localhost/xampp/
2. Buka program menggunakan tools NetBeans IDE
3. (Jika perlu) add library terlebih dahulu (DebagsPro/library) tambahkan semuanya
4. Koneksikan terlebih dahulu modules Apache dan MySql yang ada di XAMPP
5. Run Program
6. TAMPILAN AWAL:
	- akan masuk terlebih dahulu ke form login karyawan (username: fiqri, password: fiqri)
	- setelah login, maka akan langsung masuk ke form transaksi
	- lakukan transaksi
	- pilih kode barang yang akan dibeli, masukan jumlah yang diinginkan
	- jika ingin menambah barang, klik button tambah
	- setelah itu lakukan pembayaran dengan memasukan nominal uang yang diinginkan
	- setelah itu akan keluar kembaliannya (jika nominal cukup)
	- klik button cetak untuk mencetak Struk
7. Jika ingin menambah data admin/karyawan, harus terlebih dahulu login sebagai admin
8. Login dengan username: aceng, password: aceng
9. Di menu ini,akan terdapat menu item daftar karyawan, daftar admin dan data produk.
10. Fungsi dari semua menu diatas yaitu memasukan data
11. Untuk mengakses laporan keseluruhan terdapat di menu transaksi yang mana dapat diakses oleh karyawan
12. Cukup sekian program ini.
